<!DOCTYPE html>

Esta declaración define el tipo de documento como HTML5. Siempre debe ser la primera línea en un documento HTML.

<html>

La etiqueta raíz que envuelve todo el contenido de la página HTML.

El atributo lang dentro de esta etiqueta específica el idioma principal del documento

(ej: <html lang="es"> ).
<head>

Contiene metadatos sobre el documento HTML, que no se muestran directamente en la página web. Etiquetas comunes dentro de <head>:

  • <title>

    Define el título de la página que se muestra en la barra de título del navegador.

  • <meta>

    Proporciona metadatos como la codificación de caracteres ( <meta charset="UTF-8">), la descripción de la página para motores de búsqueda, y la configuración de la ventana gráfica para dispositivos móviles.

  • <link>

    Se utiliza para vincular hojas de estilo CSS externas ( <link rel="stylesheet" href="styles.css">) o iconos de página (favicon).

  • <style>

    Permite incluir estilos CSS directamente en el documento HTML.

<body>

Contiene el contenido visible de la página web, como texto, imágenes, enlaces y otros elementos. Secciones Comunes Dentro del <body>

<header>

Representa la sección de encabezado de la página o de una sección dentro de la página. Puede contener el título del sitio, el logotipo y la navegación principal.

<nav>

Define una sección de navegación con enlaces a otras páginas o secciones dentro de la página actual.

<main>

Contiene el contenido principal de la página, que es único para cada página.

<section>

Define una sección temática dentro del contenido principal. Se utiliza para agrupar contenido relacionado.

<article>

Representa un contenido independiente y autónomo, como una publicación de blog, un artículo de noticias o un comentario.

<aside>

Contiene contenido relacionado pero secundario, como barras laterales, anuncios o información adicional.

<footer>

Define la sección de pie de página de la página o de una sección. Puede contener información de contacto, enlaces a políticas de privacidad o derechos de autor.

Otras Etiquetas Importantes

<h1> a <h6>

Definen encabezados de diferentes niveles, donde <h1> es el más importante y <h6> el menos importante.

<p>

Define un párrafo de texto.

<a>

Crea un enlace (hipervínculo) a otra página o recurso

<img>

Inserta una imagen en la página

<ul> y <li>

Crean una lista desordenada (con viñetas)

<ol> y <li>

Crean una lista ordenada (numerada)

<div>

Define un contenedor genérico que se utiliza para agrupar y estilizar otros elementos

<span>

Define un contenedor en línea genérico que se utiliza para estilizar partes específicas de texto

<table>

Define una tabla

<form>

Crea un formulario para recopilar información del usuario

<input>

Define un control de entrada dentro de un formulario (ej: campos de texto, botones, casillas de verificación)
