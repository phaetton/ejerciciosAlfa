Enunciado

Existen varias herramientas online para validar que el contraste de color sea adecuado en una página web. Algunas de estas herramientas para trabajar con el color se pueden encontrar navegando por la web. Para esta actividad se utilizarán la herramienta contrastchecker https://webaim.org/resources/contrastchecker/ para la actividad 1 y Tawdis https://www.tawdis.net para la actividad 2.

Actividad 1

1. Verifique el uso de colores adecuado para que haya suficiente contraste entre el texto y el fondo. Dados los siguientes colores de letras, indique un color para el fondo que tenga suficiente contraste para visualizarse cómodamente. a. 0000ff y 5fff59 b. 0000ff y ffffff c. Para un texto de color azul (1e05ff) proponga un color de fondo donde el contraste sea adecuado.

Resultados

1. Para el color de letra 0000ff y el fondo 5fff59, el contraste es suficiente.

2. Para el color de letra 0000ff y el fondo ffffff, el contraste es suficiente.

3. Para el color de letra 1e05ff, un color de fondo adecuado podría ser ffffff.
