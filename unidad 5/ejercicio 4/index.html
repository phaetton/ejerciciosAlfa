PÁGINA WEB CON DISEÑO ADAPTATIVO ENUNCIADO (DESCRIPCIÓN) Cuando se desarrolla una página web es muy útil separar el contenido de las páginas de la apariencia. Para eso hay que utilizar las hojas de estilo CSS. Es importante conocer la sintaxis y la forma de utilizar las hojas de estilo para conseguir la apariencia deseada en nuestras páginas web. PREGUNTAS/ACTIVIDADES A REALIZAR 1. Crear una página web con un diseño adaptativo. Serán necesarias 2 hojas de estilo, una para cuando la página se muestre en un navegador con una resolución mayor de 768px y otra para cuando se muestre en un navegador de menos de 768px.Inserte en el documento html las 2 hojas de estilo de forma que se utilice una u otra dependiendo del caso. Compruebe el resultado ajustando la ventana del navegador y desde un navegador móvil. a. Cree una hoja de estilos para una vista de más de 768px de ancho. b. Cree una hoja de estilos para una vista de menos de 768px de ancho. La caja de la izquierda tiene una altura de 100px y la caja de la derecha de 200px. Su ancho es del 80% en caso de tener más de 768px y de 40% en caso de verse a menor resolución.
